Deterministic baseline

The authoritative recommendation and evidence surface. Same input yields the same deterministic output.

Runtime guardrails

Tool allowlists, budgets, redaction, deterministic fallback, and human escalation stay visible as guardrails rather than as a separate product persona.

System posture
AIDEN is not a generic LLM demo. The deterministic engine stays authoritative, the bounded assistive roles remain advisory, and the publication layer explains what happened using policy-linked evidence, traces, and review-ready artifacts.
Live-assist outputs are assistive explanations; gate authority remains deterministic and human-gated.

Deterministic decision core

The deterministic path remains authoritative.

  • Policy evaluation — Pinned rules produce pass, fail, abstain, and degrade outcomes.
  • Gate logic — Thresholds, fold behavior, and human review requirements stay explicit.
  • Delta logic — Remediation paths are derived from bottlenecks, not hidden model intuition.
  • Replay — Public-safe traces and evidence references make outcomes inspectable.

Bounded assistive overlay

Live assist is constrained by visible runtime guardrails.

  • Architect Assist — Shapes proposals, detects gaps, asks clarifying questions.
  • Reviewer Assist — Interprets policies, explains evidence, drafts remediation suggestions.
  • Guardrails — Tool allowlists, budgets, redaction, fallback, and escalation remain visible.
  • Authority — Assistive outputs remain advisory.

Guard → Gate → Review

Evidence and decisions are separated on purpose.

  • Guard — What the evidence says.
  • Gate — What the deterministic policy engine decides.
  • Review — What the human reviewer confirms, escalates, or rejects.
  • Fold behavior — Errors and unsafe live runs fall back to deterministic handling.

Publication layer

The demo, README, and docs are publication surfaces, not the scoring core.

  • FPF views — Functional, Procedural, Role-Enactor, and Module-Interface.
  • Governance views — Risk, control, compliance, and operations remain separate.
  • Public contract — Safety envelope and budgets are visible, not implied.
  • Evidence-first story — Mode, policy refs, traces, and remediation are all inspectable.
